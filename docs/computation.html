1 Computing intermediate phenotype

The predicted complex trait risk by GWAS summary statistic is used as intermediate genetic trait. LD pruning + p-value thresholding (P+T) is widely used but suffer from lack of effective SNPs.

1.1 P+T

LD pruning (\(r^2\) < 0.2) followed by p-value thresholding with various threholds optimized by validation prediction accurarcy (\(R^2\)). One implementation can be found at PRSice

1.2 LDpred

Consider LD effect in each block (\(\sim M / 3000\)) with various prior fraction of causal variants optimized by validation accuracy. One implementation can be found at here (also see paper)

Note that LDpred also includes codes for P+T strategy.

2 Testing interaction

2.1 Rationale

The epistasis between genetic background and the given locus is defined as the situation where the contribution of genotype (\(G_i\)) to liability depends not only on the genotype itself but also on the genetic background which is measured by the genetic component of the selected intermediate phenotype (\(I\)). In particular, the following model is used \[\begin{align*} \logit(\Pr(Y = 1|G_i, I, \alpha, \beta, \gamma)) &= \alpha + \mathbf{p}(I)^T \cdot G_i + \gamma I \end{align*}\]

, where \(\mathbf{p}: I \rightarrow \beta\) which takes \(I\) and maps to the effect size in terms of the contribution to log odds ratio. Note that the representation of \(G_i\) can be vector or loading depends on how the genetics is specified (see here). \(\gamma\) is the contribution of \(I\) to liability which should be universal across all genotypes and all loci. \(\alpha\) is the baseline liability of the trait which should also be share across loci.

2.2 Scenarios

The following four scenarios are considered (suppose a is risk allele):

  1. additive effect: \[\begin{align*} G_i &= \begin{cases} 0 & \text{, if genotype is AA} \\ 1 & \text{, if genotype is Aa} \\ 2 & \text{, if genotype is aa} \end{cases} \end{align*}\] , and the contribution of liability is \(p(I) G_i\), specifically \(p(I)\) is the baseline contribution function (the non-risk allele contribution).
  2. dominant(recessive) effect: \[\begin{align*} G_i &= \begin{cases} 1 & \text{, if genotype is aa or Aa (aa)} \\ 0 & \text{, if genotype is AA (AA or Aa)} \end{cases} \end{align*}\] , and \(p_i(I)\) is the corresponding contribution function of risk allele case.
  3. free effect: \[\begin{align*} G_i &= \begin{cases} (0, 0)^T & \text{, if genotype is AA} \\ (0, 1)^T & \text{, if genotype is Aa} \\ (1, 0)^T & \text{, if genotype is aa} \end{cases} \end{align*}\] , namely every genotype has a specific contribution function. Note that the effect size is respective to the reference allele (non-risk allele). The effect of non-risk allele is obserbed into \(\alpha + \gamma I\).

2.3 Interpretation under linear model

We focus on the following model \[\begin{align*} \logit(\Pr(Y = 1 | G_i, I)) &= \alpha + \beta_1 G_i + \beta_2 I + \gamma I G_i \end{align*}\]

Then, \(p(I) = \beta_1 + \gamma I\) in this case.
